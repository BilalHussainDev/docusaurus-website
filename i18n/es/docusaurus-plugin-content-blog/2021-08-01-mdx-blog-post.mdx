Las entradas del blog admiten [funcionalidades de Markdown de Docusaurus](https://docusaurus.io/docs/markdown-features), como [MDX](https://mdxjs.com/).

:::tip

Utiliza el poder de React para crear entradas de blog interactivas.

```js
<button onClick={() => alert("¡botón clickeado!")}>¡Haz clic!</button>
```

<button onClick={() => alert("button clicked!")}>Click me!</button>

:::
